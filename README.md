# My Awesome Project

基于 [VitePress](https://vitepress.dev/) 搭建的静态文档站点。

## 在线访问

https://lxhcaicai-qwq.github.io/my-vitepress/

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建
npm run docs:build

# 预览构建结果
npm run docs:preview
```

## 部署

推送到 `main` 分支后，GitHub Actions 会自动构建并部署到 GitHub Pages。
